Using LSTM to predict stock market prices for AAPL stock

Link to Github Repo: https://github.com/abdullahahsann/AAPL-Stock-Prediction-Using-LSTM

So this was just a testing project that i did to test out LSTM.

The data used is real though and is downlaoded from: https://finance.yahoo.com/quote/AAPL/history?p=AAPL.

The data is from 2017-09-20 to 2022-09-15.

First i downloaded the data as csv file from the link shared above. Then i used pandas to read the csv file.

Using matplotlib i plotted the data to see how it looks like. As it can be seen the stocks price are increasing over time.

That is going to be enough for training our model.Zooming in on just the last data points give us a more clear picture of how the data looks like.

As we are dealing with stock market data. It’s best to convert the graph into a candlestick graph.

A candlestick has four main components (open, close, high & low price) for a given time period, usually within a day. The color (usually green) indicates that the security closed higher than it opened, while the other color (usually red) indicates that the security closed lower than it opened.

For that i used potly library.

Candlestick with moving averages of 30 and 50 peroiods…
